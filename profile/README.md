# Loopback — Portable Virtual Audio Device Creator

## About Loopback

Loopback provides a complete virtual audio device environment for users who need to route audio between applications. Loopback combines virtual audio device creation, app audio capture, microphone mixing, and audio pass‑through. Loopback enables users to create virtual cables that send audio from one app to another. Loopback runs on desktop platforms. Loopback is trusted by podcasters, streamers, musicians, video editors, and audio professionals worldwide.

## Deep Dive — Virtual Audio Device Notes

Virtual devices: create virtual input/output devices, name custom
App capture: capture audio from any app (Safari, Zoom, Spotify)
Pass‑through: send audio from one app to another without recording
Mixing: combine multiple sources (microphone, app audio, system)
Monitoring: listen to virtual device output in real time
